{{'splash.updatedTo' | translate}} v3.5.0!

Many new features now available!

wb_sunny

Weather widget

The much requested, long awaited, weather widget is finally here! Get all the weather data you need at a glance with both current weather and a forecast. Like always, lots of customization available!

coronavirus

COVID-19 data widget

Now you can track COVID statistics by country on your New Tab.

access_time

Binary clock

Show off your nerd pride and read time like a machine!

textsms

Random message

You can now add multiple custom messages to a list and have a random one display on every new tab!

content_copy

Duplicate clocks

Want a second clock setup exactly the same way but with a different time zone? Duplicate it!

auto_awesome_mosaic

Place widgets anywhere

New position options for all widgets! Now you can place each widget in different locations on the screen. Place the quick links on the bottom, move the clock to the top corner, the choice is yours!

download_for_offline

Backup and restore

Backup and restore your settings with a new Export/Import option!

notes

Disable selection

Added option to disable text selection! No more accidental highlighting.

brightness_medium

Date delimiter dimming

Added option to dim the delimiter on the short date.

Minor changes

  • Many size options have greater range to select from.
  • Added extension version number to settings panel
  • Removed logo from corner
  • Adjusted some default settings
  • Improved help text for several settings
  • Translation improvements

Bug fixes

  • Fixed bug preventing patterns from displaying in certain situations.
  • Fixed occasional 404 error for background images.
  • Fixed issue where bookmarks wouldn't display when enabling bookmarks bar.
  • Fixed display bug in bookmarks bar.
